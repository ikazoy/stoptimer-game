# ぴったりストップ大会

お祭りブース向けのiPad横向きストップタイマーゲームです。子供がプリセットを選び、指定秒数ぴったりで止めることを狙います。設定とプリセットは端末ローカルに保存され、PWAとしてホーム画面に追加しておくとネットワークが不安定な会場でも遊べます。

## セットアップ

```bash
pnpm install
```

## 開発

```bash
pnpm dev
```

## 本番ビルド確認

```bash
pnpm build
pnpm preview
```

## テスト

```bash
pnpm lint
pnpm build
pnpm test:e2e
```

## 管理者画面

- プレイヤー画面右上の歯車を長押しすると管理者PIN画面が開きます。
- 初期PINは `0000` です。
- 管理者画面で、プリセットの追加、編集、削除、リネーム、JSON import/export、PIN変更ができます。

## プリセット

プリセットでは、目標秒数、カウントを表示する秒数、1等から3等の誤差しきい値、テーマカラーを設定できます。表示秒数は目標秒数以下にしてください。目標秒数と同じ値にすると、最後までカウントを表示します。

## PWA運用

会場前に、iPadで一度オンライン起動し、ホーム画面へ追加してください。その後、ホーム画面から起動できることと、オフラインでも画面が開けることを確認してから運用します。
